# UI-гайд для "Мастерская по ремонту обуви"

## 1. Цветовая схема

| Назначение | Цвет | HEX |
|------------|------|-----|
| Основной (шапка, кнопки) | Синий | `#2A5C82` |
| Основной hover | Тёмно-синий | `#1E4568` |
| Акцент (админ-панель) | Оранжевый | `#E67E22` |
| Успех (зелёный) | Изумрудный | `#27AE60` |
| Ошибка (красный) | Алый | `#E74C3C` |
| Предупреждение | Золотой | `#F39C12` |
| Текст основной | Тёмно-серый | `#333` |
| Текст второстепенный | Серый | `#777` |
| Фон страницы | Светло-серый | `#F5F5F5` |
| Фон карточек | Белый | `#FFFFFF` |

## 2. Типографика

- **Основной шрифт:** Inter, sans-serif
- **Базовый размер:** 16px
- **Заголовки:**
  - H1: 30px / 700
  - H2: 24px / 600
  - H3: 20px / 600
  - H4: 16px / 500

## 3. Сетка

- **Максимальная ширина контейнера:** 1200px
- **Отступы:**
  - От краёв экрана: 20px
  - Между карточками: 20px
  - Внутри карточек: 20px

## 4. Компоненты

### Кнопки
- Фон: `#2A5C82`
- Скругление: 4px
- Hover: затемнение + поднятие на 2px

### Карточки
- Фон: белый
- Скругление: 8px
- Тень: `0 2px 5px rgba(0,0,0,0.1)`

### Поля ввода
- Граница: `1px solid #ddd`
- Скругление: 4px
- При фокусе: обводка `#2A5C82`

### Слайдер
- Высота: 400px (desktop) / 300px (tablet) / 200px (mobile)
- Автопрокрутка: 3 сек
- Кнопки: круглые, полупрозрачные

## 5. Адаптив

| Устройство | Ширина | Особенности |
|------------|--------|-------------|
| Десктоп | > 768px | Полная версия |
| Планшет | 481–768px | Уменьшенные отступы |
| Телефон | ≤ 390px | Одна колонка, крупные кнопки |
